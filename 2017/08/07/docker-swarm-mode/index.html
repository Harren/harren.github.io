节点下线

有些时候需要维护一个节点，此时此节点可能会网络断开或者需要关机，造成节点上服务可用。使用docker node update —availability drain 将节点下线，swarm会将当前节点上的容器关闭并在其他节点上启动。当维护完成，需要上线是，将节点状态修改为active状态即可,命令如下：docker node update —availability active ，举例说明
